> This project will help us to create custom annotations in java.

## Install

```sh
mvn install
```

## Run tests

```sh
mvn test
```
### Run test for specific class
```sh 
mvn -Dtest=ClassName#testFOO test 
```
 <p>This command macthes the pattern testFoo in test class </p>
